Introduction to Interactive Programming in Python - Week 2

Week 2 introduces the basics of event-driven programming and covers the differences between local and global variables. This week's mini project is "Guess the Number", a simple game where one player comes up with a secret number and a second player attempts to guess the number. The program responds with either "Higher", "Lower", or "Correct" based on the player's guess.
